# Changelog

## 0.1.0

- Initial release as a Ruby gem
- `Legendator.configure` block for programmatic configuration
- `Legendator.translate` and `Legendator.translate_content` public API
- `Legendator.dry_run` and `Legendator.dry_run_content` for token estimation
- CLI executable (`bin/legendator`)
- OpenAI and OpenRouter provider support
- SRT parsing with UTF-8/BOM handling
- Token-aware chunking via tiktoken
- Structured JSON translation responses
